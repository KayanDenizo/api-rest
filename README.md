# api-rest

![TypeScript](https://img.shields.io/badge/-TypeScript-blue?logo=typescript&logoColor=white) ![License](https://img.shields.io/badge/license-ISC-green)

## 📝 Description

api-rest is a robust and scalable RESTful API boilerplate developed with Express.js and TypeScript. Designed for reliability and maintainability, it features seamless database integration and a comprehensive testing suite to ensure high code quality. This project serves as a solid foundation for building type-safe web services with modern backend architecture.

## ✨ Features

- 🌐 Api
- 🗄️ Database
- 🧪 Testing


## 🛠️ Tech Stack

- 🚀 Express.js
- 📜 TypeScript


## 📦 Key Dependencies

```
express: ^4.19.2
```

## 🚀 Run Commands

- **THIS_MAKEFILE_PATH**: `make THIS_MAKEFILE_PATH`
- **THIS_DIR**: `make THIS_DIR`
- **install**: `make install`
- **node_modules**: `make node_modules`
- **lint**: `make lint`
- **test-node**: `make test-node`
- **test-browser**: `make test-browser`
- **test**: `make test`
- **coveralls**: `make coveralls`
